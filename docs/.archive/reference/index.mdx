### [CLI Reference](./cli)
Complete command-line interface reference for MXCP. All commands, options, and usage examples for the `mxcp` CLI tool including:

- Quality assurance (`validate`, `test`, `lint`, `evals`)
- Operations (`logs`, `drift`, `dbt`)
